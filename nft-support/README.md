# [ECOSYSTEM CHALLENGE] Enable NFT Support

### Run Substrate Node:

```
cd substrate-node-template

cargo build --release

./target/release/node-template --dev
```

### Run Substrate Frontend:

```
cd substrate-front-end-template

yarn

yarn start
```
