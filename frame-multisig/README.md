# [INTERMEDIATE CHALLENGE] FRAME Multisig

### Run Substrate Node:

```
cd substrate-multisig-nodes

cargo build --release

./target/release/node-template --dev
```

### Run Substrate Frontend:

```
cd substrate-multisig-frontend

yarn

yarn start
```
